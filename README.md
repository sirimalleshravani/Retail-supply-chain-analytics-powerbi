# Retail Supply Chain Analytics Dashboard

## Overview
This project focuses on analyzing retail supply chain performance using Power BI. The dashboard provides a comprehensive view of sales, profit, delivery efficiency, returns, and discount impact.

The goal is to transform raw transactional data into meaningful insights that help identify profitability drivers and operational inefficiencies.

---

## Objectives
- Analyze overall sales and profit performance
- Evaluate the impact of discounts on profitability
- Monitor delivery time and shipping efficiency
- Identify top-performing regions, categories, and products
- Track return rates and their business impact

---

## Dashboard Highlights

### 1. Key Performance Indicators (KPIs)
- **Total Sales:** 2.3M
- **Profit Margin:** 12.5%
- **Average Delivery Time:** 34.6 Days
- **Return Rate:** 8%

These KPIs provide a quick overview of business performance.

### 4. Product & Category Performance
- Top categories by profit (Technology, Office Supplies, Furniture)

### 5. Delivery & Shipping Analysis

## Key Insights
- Higher discount levels (20%+) significantly reduce profit
- Majority of revenue is generated from West and East regions
- Standard shipping contributes the highest sales volume
- Technology category is the most profitable segment
- Return rate remains consistent across categories (~8%)

## Data Processing Steps
1. Imported raw retail dataset into Power BI
2. Cleaned and transformed data using Power Query
3. Created calculated columns and measures using DAX
4. Built interactive visuals and applied filters
5. Designed dashboard layout for clarity and usability
